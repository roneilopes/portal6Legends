Apresentação de Tecnologias


Netbeans
NetBeans IDE permite desenvolver aplicações web e movéis, bem como aplicativos HTML5 com HTML, JavaScrip, CSS. O IDE também fornece um grande conjunto de ferramentas para PHP e programadores C / C ++. É livre e de código aberto e tem uma grande comunidade de usuários e desenvolvedores ao redor do mundo.

PHP
PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web

CSS
Cascading Style Sheets (CSS) é uma linguagem de folhas de estilo utilizada para definir a apresentação de documentos escritos em uma linguagem de marcação, como HTML ou XML. Sua principal vantagem é a separação entre o formato e conteúdo de um documento.

HTML
HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. Atua como construtor bruto da página, sendo o CSS o principal agente de modelagem visual da página.
